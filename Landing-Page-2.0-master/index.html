This is my Coding Page

In computing, a code page is a character encoding and as such it is a specific association of a set of printable characters and control characters with unique numbers. Typically each number represents the binary value in a single byte.

What is Coding ?

Coding creates a set of instructions for computers to follow. These instructions determine what actions a computer can and cannot take. Coding allows programmers to build programs, such as websites and apps. Computer programmers can also tell computers how to process data in better, faster ways.

HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.

CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work.

JAVASCRIPT

JavaScript is an open-source programming language. It is designed for creating web-centric applications. It is lightweight and interpreted.

Future of Coding

The job market for coders is overwhelmingly a positive one and will remain so for the near future. The Bureau of Labor Statistics projects a 25% increase in jobs for software developers, testers, and quality assurance analysts between 2021 and 2031.

Is coding is needed in day-to-day life?

Coding allows humans to communicate with these devices. Modern technology such as traffic lights, calculators, smart TVs, and cars use internal coding systems.
